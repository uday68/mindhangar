### 1. **Language Selection Not Working**
**Severity**: CRITICAL
**Description**: When I try to select Hindi or Tamil from the language selector, nothing happens. The UI remains in English.
**Steps to Reproduce**:
1. Open the application
2. Go to Settings
3. Try to change language to Hindi
4. UI doesn't update

**Expected**: UI should translate to selected language
**Actual**: Stays in English

**Impact**: Makes the app unusable for non-English speakers - defeats the entire purpose of "AI for Bharat"

---

### 2. **Offline Mode Not Functional**
**Severity**: CRITICAL
**Description**: When I turn off my internet, the app shows blank screens. No cached content is available.
**Steps to Reproduce**:
1. Use app with internet
2. Turn off mobile data
3. Try to access any panel

**Expected**: Should show cached content and work offline
**Actual**: Blank screens, "Failed to load" errors

**Impact**: Unusable in rural areas or when data runs out

**Steps to Reproduce**:

**Expected**: Content in selected Indian language
**Actual**: English only

**Impact**: Non-English medium students cannot use the platform
